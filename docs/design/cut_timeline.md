# 컷 타임라인 — 구현 스펙

> 레일 슈터의 **1 컷 = Cinemachine VCam 1개 + 스폰 트리거 1세트**에 대응합니다.  
> 카메라는 **정지 컷**(적 처치 완료 시 다음 컷 진행) 방식입니다.  
> 좌표: 화면 정면 기준 **좌(L) / 중(C) / 우(R)** × **원경(遠) / 근경(近)**

Stage 1의 `_startTimes` 배열은 이 문서의 타임코드를 초 단위로 변환한 값입니다 (`Stage1CameraController.cs`).

### 설계 원칙

- **컷당 처치 목표**: 그런트 1체 4~5초, 웨이브 15~25초(여유 포함).
- **난이도 곡선**: 그런트 → 엘리트, 단발 → 동시 웨이브, 단순 약점 → 다단 QTE.
- **페널티 배치**: 스테이지 1~3에 인질/기믹, 스테이지 4는 물량+연출 클라이맥스.
