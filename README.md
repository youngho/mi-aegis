# 프로젝트: 이지스 (Project: AEGIS)

## 1\. 개요

  * **장르:** 레일 슈터 (Virtua Cop 스타일)
  * **개발 목표:** 1인 개발, 약 1시간 분량 플레이 타임
  * **핵심 컨셉:** "하나의 거대한 랜드마크 건물"을 배경으로 층별/구역별 차별화

## 2\. 세계관 및 미션

  * **세계관:** 가까운 미래, 거대 기술 기업 '넥사 코어(Nexa Core)'가 개발한 차세대 치안 AI '이지스(AEGIS)'가 테러집단에 의해 해킹당해 도시 전체를 위협합니다.
  * **주인공:** 대테러 특수부대 소속 요원
  * **미션:** 넥사 코어 본사 빌딩에 단독 침투하여, 해킹당한 AI 메인프레임을 무력화하고 인질을 구출하라.

## 3\. 스테이지 구성

# 넥사 코어 침투 작전 — 스테이지별 적 배치 상세 기획서

> 레일슈터(온레일 건슈팅) 구조 기준. 카메라는 "정지 컷(적 처치 시 진행)" 방식이며,
> 컷 하나당 표준 처치 목표시간 + 여유시간을 합산해 스테이지 플레이타임을 맞췄습니다.
> 좌표 표기: 화면을 정면에서 봤을 때 **좌(L) / 중(C) / 우(R)** × **원경(遠) / 근경(近)**

### 설계 원칙 메모
- **컷당 처치 목표시간**: 그런트 단독은 1체당 약 4~5초, 웨이브 형태는 여유시간 포함 15~25초 단위로 산정했습니다.
- **난이도 곡선**: 스테이지가 진행될수록 (그런트→엘리트 그런트), (단발 등장→동시다발 웨이브), (단순 약점→다단계 QTE)로 상승.
- **인질/기믹 페널티**는 스테이지 1~3에 배치해 긴장감을 주되, 최종 스테이지(4)는 물량+연출 중심으로 전환해 클라이맥스를 강조했습니다.
- 실제 개발 시 각 "컷" 단위가 곧 카메라 웨이포인트 1개 + 트리거 볼륨 1개에 대응되도록 설계했습니다.
